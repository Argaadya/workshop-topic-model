What Makes Text Data Different?

Messy and unstructured

Unlike your typical tabular data or spreadsheet, text data is unstructured and very messy to deal with. Different text will have different length and generally text data has bigger size than structured data.

Requires heavy cleansing

Text data must be cleansed first before being ready to be used since not all characters hold the same value.

Human are complicated creatures

Human has the ability to give different context to the same sentence (compliment, sarcasm, irony, etc.), making text analysis quite challenging.

Language-dependent

Different language can have different rule and different text processing steps.

What Makes Text Data Valuable?

They are everywhere

Official document, news, magazine, social media, online marketplace; text data is everywhere as long as people use language to communicate.

Useful for Many Purpose

The large field of Natural Language Processing that also deal with text data can be applied for many use case, including:

  • Sentiment analysis
  • SPAM email detection
  • Text summarization
  • Named-entity recognition
  • Text generation
  • Topic modeling

Topic Modeling

Finding hidden topic inside a collection of document

Topic modeling attempt to cluster our text into distinguishable cluster/topics to gain insight from large collection of text data.

Use Case
  • Automatic Labeling
  • Discover different topic in large corpus of document
  • Sentiment Analysis
  • Understanding Stance and Polarization in Social Media
  • Identify new innovation/discovery in scientific research paper
  • Document Classification
  • Recommender System

Why R?

Built by statistician

R is created for the purpose of data analysis and as such, is different in nature from traditional programming languages.

Plenty of open source libraries

R libraries extend R graphical abilities, and adds out-of-the-box functionalities for many statistics and machine learning algorithm.

Rstudio is great for reporting

Rstudio as the IDE of R provide many features that significantly improve the data analysis process and can generate many great variety of output. RStudio also provide great integration with other language, especially python.
